# AMD Developer Hackathon - ACT II 🚀

An AI-powered Multi-Agent system built with CrewAI to automate complex research and target location identification tasks. 

## 📌 Project Overview
This project leverages **CrewAI** and advanced LLMs to coordinate specialized AI agents. The agents work collaboratively to analyze data, execute task workflows, and deduce precise target locations based on user queries.

### 🤖 Core Architecture (Our Crew)
*   **Research Agent:** Analyzes the initial query and gathers crucial context.
*   **Analyst Agent:** Filters the information, handles logical deduction, and outputs the final result.

---

## 🛠️ Tech Stack
*   **Framework:** CrewAI
*   **LLM Provider:** Fireworks AI (DeepSeek-V3)
*   **Frontend (MVP Demo):** Streamlit
*   **Language:** Python 3.10+
